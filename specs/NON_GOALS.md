# Non-Goals

## Product Non-Goals

- AssessmentOS is not a generic AI exam generator.
- AssessmentOS is not a consumer worksheet toy.
- AssessmentOS is not a random copyrighted web scraper.
- AssessmentOS is not a marketplace-first product in MVP.

## MVP Non-Goals

- Full OCR
- Full AI pipeline
- Full PDF/DOCX rendering
- Approval workflow
- Analytics
- Billing
- Marketplace
- Multi-campus enterprise controls

## Engineering Non-Goals

- No provider lock-in in UI components.
- No unreviewed dependency sprawl.
- No speculative enterprise architecture before MVP behavior is validated.
- No generated UI clutter.
- No large implementation PRs that mix unrelated product areas.
